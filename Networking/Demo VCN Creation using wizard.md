# Create the VCN
Go to Virtual Cloud Network > Actions > Start VCN Wizard 
![[Pasted image 20260307212112.png]]

In the VCN Wizard we can see the 2 net structures that has
![[Pasted image 20260307212235.png]]
	We need the left structure

Follow the form to create the network
![[Pasted image 20260307212506.png]]
	It will need a name and an IP 

From this IP it will create 2 subnets
![[Pasted image 20260307212607.png]]

**And that will be all we already would have our VCN**
![[Pasted image 20260307212654.png]]
